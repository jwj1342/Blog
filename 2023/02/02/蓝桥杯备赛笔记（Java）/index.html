环境学习

Eclipse的相关有用的东西。

快捷键

  • Crtl ++或者Crtl --：是字体的放大缩小

  • Crtl + Shift + L ：打开快捷键提示菜单

  • Crtl + F11 ：RUN代码

  • Ctrl + 1 ：快速修复（可以解决很多问题，比如import类、try catch包围等）

  • ctrl+d：删除当前行

  • Ctrl + Shift + F ：格式化当前代码

  • Alt+Shift+R ：重构名称（重命名方法名、属性或者变量名）

  • Alt+Shift+M ：重构方法（抽取代码到方法）

  • Alt+Shift+Z ：重构的后悔药（Undo）

  • Ctrl + z 结束流的重定向
